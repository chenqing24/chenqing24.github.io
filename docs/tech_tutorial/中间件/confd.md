# Confd 服务配置中心

## 安装

在本地mac上验证通过

```bash
# 下载到专用目录
cd /Users/jeff.chen/opt/confd
wget https://github.com/kelseyhightower/confd/releases/download/v0.16.0/confd-0.16.0-darwin-amd64
mv confd-0.16.0-darwin-amd64 confd
# 设置环境
export PATH="/Users/jeff.chen/opt/confd:$PATH"
# 验证
confd -help
```
